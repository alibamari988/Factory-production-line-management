---

</div>📖 معرفی پروژه

سیستم مدیریت خط تولید کارخانه یک سامانه جامع برای مدیریت فرآیندهای تولید، انبار، فروش، مواد اولیه و تأمین‌کنندگان است.

هدف این پروژه، تحلیل و طراحی یک سیستم نرم‌افزاری برای مدیریت یک کارخانه تولیدی است که مدیر تولید بتواند با استفاده از اطلاعات فروش، موجودی انبار و ظرفیت خطوط تولید، بهترین تصمیم را برای برنامه‌ریزی تولید اتخاذ کند.

در این پروژه هیچ زبان برنامه‌نویسی استفاده نشده و تمرکز اصلی بر تحلیل سیستم و طراحی UML است.

⚙ زیرسیستم‌ها

📦 ۱. مدیریت انبار

قابلیت‌ها

- تعریف انبار
- ثبت ورود کالا
- ثبت خروج کالا
- مدیریت موجودی
- گزارش موجودی محصولات
- گزارش موجودی مواد اولیه

---

🧱 ۲. مدیریت محصولات و مواد اولیه

امکانات

- تعریف محصول
- تعریف مواد اولیه
- ثبت فرمول ساخت (BOM)
- تعیین مقدار مواد اولیه موردنیاز برای هر محصول

---

🚚 ۳. مدیریت سفارش مواد اولیه

امکانات

- تعریف تأمین‌کننده
- تعریف مواد قابل عرضه
- ثبت سفارش خرید
- پیگیری سفارش
- انتخاب بهترین تأمین‌کننده

---

🏭 ۴. مدیریت خط تولید

امکانات

- تعریف خطوط تولید
- تخصیص محصول به خط تولید
- برنامه‌ریزی تولید
- مدیریت وضعیت خطوط تولید

---

💰 ۵. مدیریت فروش

امکانات

- ثبت فروش
- گزارش فروش
- مشاهده آمار فروش
- پیش‌بینی فروش آینده
